# Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl+B` | Explorer切替 |
| `Ctrl+`` | Terminal切替 |
| `Ctrl+Shift+P` | Command Palette切替 |
| `Ctrl+Shift+R` | レイアウトリセット |
| `Ctrl+S` | ファイル保存 |
| `Ctrl+Shift+T` | 新規PMターミナル |
| `Ctrl+Shift+W` | 新規Workerターミナル |
| `Ctrl+Shift+Q` | 緊急停止 |

## Configuration

- `config/shortcuts.json` - カスタマイズ可能
- `config/settings.json` の `layout` - レイアウト永続化
